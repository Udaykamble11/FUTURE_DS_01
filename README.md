# Global E-Commerce Sales Analysis

# Project Overview
This project analyzes a year of sales data for a UK-based online retailer to identify revenue trends, top-performing products, and international growth opportunities. 

Tools Used: Python (Pandas) for data cleaning, Power BI for data modeling and visualization.

# Key Insights & Recommendations
1. Seasonality: Revenue heavily spikes in November, requiring marketing campaigns and inventory stockpiling to be finalized by late September.
2. Product Concentration: The 'Regency Cakestand 3 Tier' is the dominant revenue driver. 
3. International Expansion: Outside the UK, the Netherlands and EIRE (Ireland) represent the most lucrative international markets.

# Technical Process
* Data Cleaning: Used Python to handle missing Customer IDs, remove canceled orders (negative quantities), and engineer a new `Revenue` column.
* Data Modeling: Created DAX measures in Power BI for Total Revenue, Total Orders, and Average Order Value (AOV).
* Visualization: Built an interactive dashboard featuring KPI scorecards, time-series analysis, and geospatial mapping.
